### （1）制作サイトのタイトル

**ChatGPT活用16タイプ診断**

### （2）制作サイトの説明（40文字程度）

AIの活用傾向を診断し、ChatGPTをより効果的に使うための最適なパーソナライズ設定を提案する診断サービスです。

### （3）工夫した点・こだわった点

診断結果の共有機能として、Xへの共有と通常の共有機能を実装しました。Web Share APIが利用できるかを判定し、利用できない環境ではX共有へ切り替えることで、デバイスやブラウザの違いに左右されず、診断結果を共有できるようにしました。

### （4）難しかった点・次回トライしたいこと（または機能）

FirebaseのAPIキーは公式ドキュメントで公開リポジトリに含めても問題ないとされていたため、実際にGitHubへ公開しました。しかし、その後警告メールが届いたことで不安を感じ、リポジトリを削除してプロジェクトを作り直しました。APIキーの役割やFirebaseのセキュリティルールについて理解を深め、適切なアクセス制御を実装したうえで、自信を持って公開できるようにしたいです。
